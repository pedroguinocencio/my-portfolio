# Pedro Inocêncio Portfolio

Projeto estático organizado por responsabilidade.

## Estrutura

```text
.
├── index.html
├── assets/
│   └── images/
│       └── profile.png
└── src/
    ├── scripts/
    │   └── main.js
    └── styles/
        └── main.css
```

## Convenção

- `index.html`: estrutura da página e conteúdo.
- `src/styles/main.css`: identidade visual, responsividade e componentes visuais.
- `src/scripts/main.js`: interações, tema, idioma, carrossel e modal.
- `assets/images`: imagens usadas diretamente pelo site.

O portfólio em PDF é exibido pelo preview do Google Drive no modal, então não há cópia local pesada do arquivo no projeto.
